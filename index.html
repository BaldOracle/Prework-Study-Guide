HTML

  • The head element contains information about the webpage and cannot be seen on the page
  • The body element represents the visible content shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS order Margin - Border - padding - content

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -A: Command to commit all files to working branch
  • git add -m "message": add message with commit
  • git pull origin main: pull original from github before merging
  • git push origin xxx: xxx name of file being pushed
